# JOB QUEST ROGUE -クリスタルとまおう-

FF5風ジョブシステム × クロノトリガー風ATBバトルのローグライクRPG。
ブラウザでそのまま遊べます (スマホ対応・タッチ操作):

**▶ https://ryukou.github.io/jobquest-rogue/**

- 毎回レベル1で自動生成ダンジョン (B1F〜B10F) に挑む
- 全滅してもジョブのABPだけは引き継がれる
- B5Fでロボを解放、B10Fのまおうを倒せばクリア

操作: 矢印キー移動 / Z:けってい / X:メニュー (スマホは画面の仮想パッド)

pygame-ce + pygbag (WebAssembly) 製。
