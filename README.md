# JOB QUEST ROGUE -クリスタルとまおう-

FF5風ジョブシステム × クロノトリガー風ATBバトル × Slay the Spire風の
ルート選択ローグライクRPG。ブラウザでそのまま遊べます (スマホ対応):

**▶ https://ryukou.github.io/jobquest-rogue/**

- 最初に画面を1回タップするとゲームが始まります (音もここで有効になります)
- フロアごとに「そうぐう / きょうてき / たからばこ / みせ / イベント」から進路を選択
- 戦闘後は全回復。報酬で 武器・おまもり・ひでんのしょ を集めて強くなる
- 全滅してもジョブのABPだけは引き継がれる。B10Fのまおうを倒せばクリア

## 操作 (すべてタップでOK)

- **1回タップ = 即けってい**。カード・コマンド・敵をそのままタップ
- 戦闘では **敵を直接タップ**して攻撃対象を選ぶ
- 右下の **もどる** ボタンでキャンセル / メニューを閉じる
- スマホは **よこもち推奨**。PCでは 矢印キー / Z / X も使えます


pygame-ce + pygbag (WebAssembly) 製。
